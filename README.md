# uniApp-QQ
大一下学期期末作业/项目test


### 小程序名称：uniApp-扣扣

开发工具：HbuilderX  (3.2.9.20210927)

云数据库：uniCloud

数据库设计：openDB



### 使用：

- 导入项目到 Hbuilder X

- 创建uniCloud云开发环境

- 导入云数据库
  - [动态页](CloudDatabase/dongtai.json)
  - [消息](CloudDatabase/message.json)
  - [通讯录](CloudDatabase/tongxunlu.json)

- 关联云服务空间或项目
